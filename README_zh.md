# k8s-toolbox
本仓库提供了在 K8s 集群中安装一些工具 & 平台的部署代码, 支持快速上手进行 K8s 集群中工具 & 平台 部署.

## 如何使用
```
git clone https://github.com/VelocityLight/k8s-toolbox.git
cd k8s-toolbox
kubectl apply -f <folder>/
```

## 工具
```
.
├── matomo                          # 部署 matomo 平台用于快速分析网站访问量等情况.
├── README.md
└── README_zh.md
```
